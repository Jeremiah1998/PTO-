NET Pay Program

Simple NET pay calculator. It takes your GROSS Pay and then deducts Federal, State, and FICA. It has the option to deduct any additional amount from your check. Then it displays your NET pay.

  1. Start
  2. Declare grosspay
  3. Display "What's your gross pay per year"
  4. Input real grosspay
  5. Declare FED
  6. Delacre State
  7. Declare FICA
  8. Set ADD = 0
  9. Set Fed = grospay*.15
  10. Set State = grosspay*.05
  11. Set FICA = grosspay*.0762
  12. Set NET = grosspay - (Fed + State + FICA + ADD)
  13. Display "Your net pay is" + NET + "."
  14. END
